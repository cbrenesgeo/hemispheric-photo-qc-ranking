# Historial de cambios

## [0.5.0] - 2026-07-20

### Añadido

- Flujo principal consolidado para control de calidad y ranking.
- Aplicación consistente de máscara circular.
- Orientación horizontal mediante copias de trabajo.
- Validación de presencia y proporción del componente cielo.
- Ranking multicriterio configurable.
- Mecanismo de rescate para completar el top de candidatas.
- Procesamiento del top con `hemispheR`.
- Auditoría de estabilidad de LAI, DIFN y cobertura.
- Selección final con niveles de revisión.
- Script independiente para auditoría visual del top 5.
- Paneles JPG con fotografía original, canal de proceso, binarización y métricas.
- Documentación general y por script.

### Notas

Esta versión representa una consolidación metodológica. Los umbrales deben validarse antes de aplicarse a nuevas cámaras, lentes, coberturas o protocolos.
